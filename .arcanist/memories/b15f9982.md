Stripe requires PaymentIntent amounts in the smallest currency unit (cents for USD, paise for INR). When converting between currencies for Stripe, use Math.round() on the converted amount to ensure it's an integer — Stripe rejects non-integer amounts. Also, different currencies require different PaymentIntent configurations: USD can use `automatic_payment_methods: { enabled: true }` while INR with UPI requires explicit `payment_method_types: ["upi", "card"]` instead of automatic_payment_methods.
